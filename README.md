# Greywater Reuse AI

AI-based greywater reuse simulation and sustainability analytics platform for apartment-scale water infrastructure.

## Project Goal

This project estimates how much greywater an apartment building can generate, store, and reuse for non-potable purposes such as toilet flushing.

It uses software simulation and data analytics instead of physical sensors, making it a zero-capital starting point for smart water infrastructure research.

## Features

- Synthetic apartment water usage data generation
- Greywater tank simulation
- Overflow and shortage estimation
- Freshwater savings calculation
- Jupyter notebook analysis
- Streamlit dashboard prototype

## Tech Stack

- Python
- Pandas
- NumPy
- Matplotlib
- Streamlit
- scikit-learn
